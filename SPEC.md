# 語学学習ストリーク記録【無料】無料Webツール - 技術仕様書

## 概要

**サービス名**: Language Learning Streak
**バージョン**: 1.4.0
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/language-learning-streak/

毎日の語学学習時間・内容を記録して継続日数（ストリーク）を管理。登録不要・完全無料でご利用いただけます。

## v1.4.0 グラフ・ビジュアライゼーション追加 (2026-06-10)

### 言語別統計グラフ化（棒グラフ）（実装）
- 言語ごとの累計学習時間を棒グラフで可視化（Chart.js）
- 言語名と学習時間（時間単位）を直感的に比較
- 複数言語学習者が学習バランスを一目で把握可能

### 週間学習進捗スコアゲージ（実装）
- 週間学習目標に対する進捗を大きなパーセンテージで表示（例: 60%）
- プログレスバー（既存）に加えて、数値スコアで目標達成度を可視化
- 目標達成時は「✅ 今日の学習は記録済みです！」バナーでモチベーション向上

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 記録したい項目を入力フォームに入力する
2. 「学習を記録する」ボタンをクリックする
3. 記録一覧で過去のデータを確認する
4. 言語フィルターで特定言語の記録を絞り込む
5. 言語別統計で各言語の学習進捗を確認する
6. データをエクスポート（CSV/JSON）してバックアップする

## v1.3.0 UX改善 (2026-06-07)

### カスタム言語入力（実装）
- 「その他（自由入力）」選択時にテキスト入力フィールドを表示
- ポルトガル語・ロシア語・イタリア語を固定リストに追加
- カスタム入力した言語名は統計・フィルターにもそのまま表示

### 学習時間クイック選択ボタン（実装）
- 15分・30分・45分・60分・90分のチップボタンでワンタップ入力
- アクティブ選択を視覚的にハイライト表示

### カレンダー月別ナビゲーション（実装）
- 直近28日固定から月単位表示（1日〜末日）へ変更
- 「‹ 前月」「翌月 ›」ボタンで任意の月を確認可能
- ホバーツールチップに言語・学習時間を表示（例: 英語 30分 / 中国語 45分）

### 週間学習目標と進捗表示（実装）
- 週に何日勉強するかの目標を1〜7日で設定（localStorage保存）
- 今週（月曜〜日曜）の学習日数をプログレスバーと数値で表示
- 目標達成時は「✅ 達成！」表示でモチベーションアップ

## よくある質問（FAQ）

**Q: 語学学習ストリーク記録は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。
